Display : Flex

Flex is short for the Flexbox Box Layout module. Flexbox is a method for arranging items in rows or columns . Flexbox makes it easier to design a Flexbox responsive layout , without using float or positioning.

  • row-nowrap
  • column-wrap
  • column-wrap-reverse
  • column-nowrap
